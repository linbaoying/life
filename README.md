## 诗词
- [《新年旧岁》](#诗词)
> **旧岁新年**  
> **林抱影**  
> 长天月清浅，江枫歌未眠。  
> 绮陌迎新岁，孤馆度旧年。  
> 名利长萦绊，烟水程何限。  
> 停灯对窗畔，向晓影无眠。
